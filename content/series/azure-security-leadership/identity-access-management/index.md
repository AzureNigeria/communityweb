## Session Abstract

Identity is the new security perimeter in the cloud. This session gives a clear, practical path to building IAM foundations for Zero Trust in Azure without slowing the business. You will understand how identity attacks happen and what it takes to design secure access with Microsoft Entra ID.

The session blends technical fundamentals with leadership-level decision guidance, helping teams reduce identity-driven risk while keeping productivity high. Expect a focus on patterns you can apply immediately in Azure environments.

### What you'll learn:

- Why identity-led controls are critical for cloud security.
- Core IAM concepts and the most common attack patterns.
- How to apply Zero Trust principles to Azure access design.
- Practical ways to secure access while keeping teams productive.
